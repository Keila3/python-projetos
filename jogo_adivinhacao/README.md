
# Jogo da Adivinhação

Breve e objetivo: jogo simples em terminal onde o jogador tenta adivinhar um número aleatório gerado pelo computador.

**Objetivo:** Adivinhar o número correto em menos tentativas possível.

**Como jogar:**
- Execute `python main.py`.
- O programa gera um número (ex.: entre 1 e 100).
- Insira palpites até acertar; o jogo informa se o palpite é maior ou menor que o número alvo.

**Requisitos:**
- Python 3.x

**Arquivo principal:**
- `main.py` — ponto de entrada do jogo.
